🇺🇸 今日重大事件 #1 — 美伊停火協議實質破裂，美軍再次空襲伊朗
Trump宣告伊朗MOU「告終」，美軍對伊朗多處目標展開新一輪打擊，油價與殖利率同步走升
7月8日（週三），美國總統Trump宣告與伊朗的停火備忘錄「基本上已結束」，並稱美方將繼續對伊打擊。美軍中央指揮部確認已完成對伊朗多處目標的空襲行動，理由是伊朗船隻在荷姆茲海峽攻擊三艘商業船舶。此舉推升WTI原油單日飆漲+5.79%至74.52美元，10年期美債殖利率同步走高。與此同時，Fed 6月FOMC紀要顯示部分官員討論在通膨未受控時進一步升息的可能性，油價飆升強化鷹派憂慮。市場67.3%個股當日下跌，道瓊重挫577點；摩根大通跌2.5%、Visa跌1.3%、Alphabet跌1.4%。IMF亦上調2026年油價預測達32%、全球通膨至4.7%。（來源：TheStreet、CNBC、TradingEconomics，2026-07-08）
🇹🇼 今日重大事件 #2 — 台股7日高低震幅1,535點，終場重挫千點；台積電獨撐紅盤
台股昨日盤中衝上46,967點後急殺，終場收45,479點大跌1,077點；三大法人大幅淨賣超937億元
7月7日（週二），台股開高後遭遇三星財報公布利空（三星Q2雖創歷史盈利紀錄，但部分市場預估落空）引發亞洲半導體賣壓擴散，盤中一度急殺至45,432點，高低震幅高達1,535點，終場收45,479.11點，創史上第8大收盤跌點。外資大量調節高價半導體股，聯發科一度跌至3,985元（盤中跌幅逾115元），日月光投控、穎崴、矽力-KY等皆跌停。惟台積電（2330）憑藉TSM ADR溢價支撐（+13.77%）、月營收YoY+30.1%正向預期，終場逆勢小漲25元收2,465元。三大法人合計淨賣超約937.82億元（外資-547.31億、投信+96.83億、自營-487.33億）。財信傳媒董事長謝金河分析此波震盪主因是「全球市場面臨去槓桿壓力，全世界都在陪南韓去槓桿」，而非AI題材泡沫破裂。（來源：ETtoday、Yahoo股市、聯合新聞網，2026-07-07）
🇺🇸 今日重大事件 #3 — Broadcom大漲+4.83%、NVDA+3.65%，AI晶片需求再獲驗證
Broadcom擴大與Apple合作協議至2031年，中企計畫增購NVIDIA H200，AI需求面利多聚焦

🇰🇷 今日重大事件 #4 — 三星Q2創歷史紀錄，SK海力士ADR(SKHY)7/10 Nasdaq掛牌倒數

本週主軸為「三星財報+SK海力士ADR掛牌」構成的半導體雙催化劑，疊加美伊停火破裂帶來的能源通膨恐慌，市場呈現明顯板塊分化：AI晶片（NVDA+3.65%、AVGO+4.83%）走強，但能源敏感的金融與工業股（Dow -1.09%）遭拋售。三地受傷程度排序：韓股前日（7/7）最重（KOSPI引發去槓桿連鎖效應）> 台股（-2.31%千點急殺）> 美股（Nasdaq逆勢+0.20%）。目前判斷：本次急殺性質偏向洗盤70% / 出貨30%，因AI基本面並無惡化（三星Q2破紀錄、CSP Capex持續上修），主因是韓國槓桿ETF去槓桿引發的技術性拋售，不代表AI題材信仰動搖。最近驗證點：7/10 SKHY掛牌表現（F確認後）及7/16台積電法說會（F確認後，判斷升降級關鍵）。

🔗 事件鏈分析

催化劑①（負向）：韓國散戶大量湧入兩倍槓桿ETF（三星/SK海力士），韓國金融監督院強制去槓桿 → 三星、SK海力士跌停式拋售 → KOSPI暴跌近20%（整個下跌期間） → 亞洲半導體情緒共鳴 → 台積電遭連帶拋售 → 台股千點急殺。

催化劑②（負向）：美伊停火協議破裂、WTI油價飆升5.79% → 通膨預期重燃 → FOMC紀要顯示部分官員傾向升息 → 美債殖利率上行 → 信用敏感板塊（金融/工業）遭殺 → Dow重挫577點。

催化劑③（正向）：三星Q2營業利益89.4兆韓元破紀錄 + NVIDIA H200中企採購回溫 + AVGO與Apple合作延長至2031年 → AI晶片需求基本面再獲驗證 → NVDA、AVGO、SMH逆勢走強。

背離警示（I）：台積電在台股整體急跌時收漲+1.02%，TSM ADR溢價+13.77%，顯示外資對台積電基本面信心未動搖，此「價格走勢與大盤背離」是洗盤信號而非出貨訊號。若台積電7/16法說後月營收年增仍維持>30%，洗盤概率將升至80%。
