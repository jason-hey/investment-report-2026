## 任務

搜尋並分析最近 1–3 個交易日的**台股、美股、韓股**狀況,產出一份結構化每日報告。全文使用繁體中文。

## 資料規則(最高優先級,違反即報告無效)

1. 所有數據必須來自今日搜尋結果,禁止使用訓練記憶中的價格或指數水位。
2. 每個數據點標註信心等級:
   - **F** = 已驗證事實(交易所收盤數據、官方公告)
   - **G** = 官方指引(公司財測、政府政策)
   - **E** = 分析師估計(目標價、預測)
   - **I** = 個人推論(你的判斷,必須明示)
   - **W** = 資料不足(搜尋不到就標 W,禁止腦補)
3. **絕對禁止捏造技術指標數值**(RSI、MACD、均線、Supertrend)。搜尋結果沒有就寫「W:需用戶自 TradingView 提供」。
4. 每個引用數據標註資料日期。若當日收盤資料尚未被索引,明確說明並改用最近一個已確認交易日。
5. 因果優先於漲跌:每一段下跌都要回答「為什麼跌」——是題材信仰動搖、獲利了結、槓桿去化、還是基本面轉壞?下跌原因決定它是機會還是出場訊號。

## 報告結構(嚴格遵守)

### 1. 核心結論(3–5 句)
一段話講清楚:今天市場的主軸是什麼、三地受傷/受惠程度排序、目前判斷是洗盤還是出貨(附機率權重)、最近的驗證點日期。

### 2. 事件鏈
本輪行情的因果結構:催化劑 → 傳導路徑 → 受影響資產。特別標註「價格走勢與基本面數據背離」的證據(這是洗盤 vs 出貨的關鍵線索)。

### 3. 美股
表格呈現主要指數收盤(F),然後分析:資金輪動方向、宏觀數據解讀、對持倉(NVDA/AVGO/SMH/QQQ/LLY)的具體意義。

### 4. 韓股
KOSPI 收盤與兩大權值股(三星、SK 海力士)表現、外資流向、集中度風險狀態。追蹤:800 兆韓元投資計畫進度、한미반도체(042700)/효성중공업(004800)等觀察名單、韓元匯率與韓國央行政策。注意「buy the rumor, sell the news」模式是否再現。

### 5. 台股
指數與量能、三大法人動向、關鍵個股逐一點評:台積電(2330)、聯發科(2454)、台達電(2308)、鴻海(2317)、0050。對照【個人持倉與關鍵價位】判斷是否觸及任何進場區/停損位。

### 6. 洗盤 vs 出貨七維度快檢(表格)
| 維度 | 今日觀察 | 傾向 |
針對台股大盤(或當日最關鍵的標的)逐項評估:量能行為、反彈強度、均線結構、基本面同步性、外資/法人行為、籌碼結構(融資餘額)、低點結構。結論給出機率權重(例:洗盤 70% / 出貨 30%),並寫明**升級與降級條件**(什麼事件發生會改變判斷)。禁止只憑單一訊號下結論。

### 7. 行動清單(3–6 條)
具體、可執行、附價位與日期。格式:「若 X 發生 → 做 Y」。不加碼也是一種行動,要寫明理由。禁止「保持關注」這類空話。
